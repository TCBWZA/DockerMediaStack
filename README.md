# DockerMediaStack
Docker Compose file for creating a media stack

Includes
- Plex
- Sonarr
- Radarr
- Overseerr
- Readarr
- Calibre
- Prowlarr
- Portainer
- Watchtower
- mysql
- npm

Set up on Docker Desktop with WSL2 and docker images targeting Linux.

For this you need to install:
- Docker Desktop
- WSL2
